## ٢. إيه الـ Brand Reveal (الشغل الحالي)

تتابع سينمائي (زي intro الشركات الكبيرة — Anthropic/Apple) بيظهر **بعد ما المستخدم
يضغط أيقونة Problem** في صفحة الـ intro (شاشة iPhone وهمية) و**قبل** صفحة الـ login.

**الفلسفة:** كشف تدريجي **متتابع** (مش متزامن) — كل مرحلة تبني توقّع للي بعدها.
بطيء ومهيب. اللون يتدرّج من ice أبيض → وميض cyan في لحظة الذروة (الوضوح).

### المراحل الحالية (٨.٣ ثانية):
```
١. نقطة وعي (cyan) تنبض في الظلام
٢. الجفون تفتح — شكل almond، تدرّج ice→cyan، توهّج
٣. حلقات المتاهة ترسم للداخل واحدة واحدة + جدران radial
٤. مسار الحل (solve path) يتتبّع نفسه للمركز — خيط متوهّج + شرارة بالمقدمة
٥. البؤبؤ يشتعل + نقطة وعي تدور حواليه (orbit)
٦. حروف "Problem" تتجمّع واحد واحد (خط Cormorant Garamond)
٧. "إيمان" تظهر تحتها (خط DM Mono)
٨. ✦ وميض cyan — لحظة الوضوح، ثم خروج
```

## ٣. العين الأصلية للموقع (لازم تطابقها — دي الـ DNA)

العين في الموقع موجودة في الـ hero V1: SVG `id="heroEyeSvg"` كلاس
`hero-eye-svg qa-watching`، viewBox `0 0 900 750`، مركزها `translate(450,340)`.
دي **عين/متاهة فلسفية** (مش عين تشريحية بسيطة)، كلها stroke-based بـ ice/cyan،
مفيش fills غير أقنعة بلون الخلفية بتفتح فتحات في الحلقات:

- **الهالة:** radialGradient `#hAura`.
- **الجفون:** علوي `M -290 0 Q 0 -185,290 0` (توهّج blur) + الجفون المرئية
  `M -290 0 Q 0 0,290 0` بـ linearGradients `#hU` (علوي) / `#hL` (سفلي). بترسم عبر
  stroke-dashoffset. فيه "tear duct" (مجرى دمع) صغير.
- **حدود القزحية:** ellipse rx=120 ry=80.
- **المتاهة (القزحية):** ٥ حلقات متحدة المركز نصف أقطارها **95/75/55/38/22**
  (كلاسات maze-ring-1..5)، جدران radial قصيرة كتير (lines)، أقواس فجوات بلون
  الخلفية بتفتح ثغرات، و**مسار حل متوهّج** (`#heroSolvePath`، يرسم عبر
  stroke-dashoffset، توهّج بفلتر `#hG2`) بيتتبّع من الحلقة الخارجية للمركز.
- **حلزون ذهبي + ٨ بتلات Fibonacci** (ذهبي `rgba(232,196,104)`) — بيظهر عند
  "solved" (كلاس .qa-embody-spiral). لمسة ذهبية اختيارية للتتابع.
- **البؤبؤ:** `#heroPupil` r=12 (توهّج)، نقطة داخلية بلون الخلفية r=4، نقاط highlight
  بيضا. حلقات ripple (.se-rip) + حلقات صدمة (.shock-ring).
- **بجانب العين:** badge أسئلة/أجوبة (`#qaBadgeGroup` عند translate(640,180)) فيها
  نجمة octagram بزوايا ذهبية (مرحلة ✦ WATCH)، علامة "?" حلزونية، إلخ. العين عندها
  ٤ مراحل بتتبدّل بكلاسات على الـ svg:
  qa-watching(✦) → qa-searching(?) → qa-found(!) → qa-solved(✓).
  **الـ badge ده إضافة منفصلة بجانب العين — على الأرجح مش محتاجينه في التتابع.**

**الـ DNA اللي لازم يتكرر للإحساس الاحترافي:** حلقات متاهة متحدة المركز، مسار حل
متوهّج بيرسم نفسه، بؤبؤ بينبض بـ highlight، تدرّجات ice→cyan، توهّج Gaussian ناعم،
easing بطيء غير مستعجل.

## ٤. إزاي الكود متعمل (Architecture)

التتابع = محرّك **canvas** اسمه `const BrandReveal = (function(){ ... })();`
(IIFE بترجّع `{ play(onDone), skip() }`).

- بيرسم بـ `draw(ctx,W,H,t)` حيث `t` بالملي ثانية من بداية التشغيل.
- التوقيت في object اسمه `T` (مراحل: dot/iris/rings/solve/pupil/word/name/flash/hold)
  و `TOTAL = 8300`.
- helpers جوّه المحرّك: `eio` (ease-in-out), `eoc` (ease-out-cubic), `clamp`,
  `lerp`, `seg(t,[a,b])` (بترجّع تقدّم 0→1 لمرحلة).
- هندسة المتاهة ثوابت: `RINGS=[95,75,55,38,22]`, `WALLS=[[angleDeg,fromR,toR],...]`,
  `GAPS=[[angleDeg,R],...]`.
- **⚠️ تحذير مهم:** المحرّك لازم يعرّف `lerp` بتاعه **جوّاه**. فيه `lerp` global بس
  هو **جوّه closure تاني ومش في scope** — لو اعتمدت عليه هيطلع
  `ReferenceError: lerp is not defined` والكانفس يفضل أسود. ده حصل قبل كده واتصلح.

### مكانين للمحرّك (نفس الكود بالظبط في الاتنين):
1. **`brand_reveal_standalone_v2.html`** — ملف مستقل للمعاينة. بيشتغل أوتوماتيك عند
   الفتح (`window.load` → `document.fonts.ready` → `BrandReveal.play`)، فيه زر
   "إعادة" + ضغط أي مكان = تخطّي. ده اللي إيمان بتفتحه عشان تشوف وتقيّم.
2. **`Ander_57_real_eye_reveal.html`** — الموقع كامل. المحرّك موجود **قبل**
   `function openProblemFromIntro()` مباشرة. والربط:
   `openProblemFromIntro()` → fade out intro → `BrandReveal.play(proceed)` →
   `proceed()` بتنادي `showLogin()`. فيه overlay `#brandReveal` + `#brandCanvas`
   + كلاس CSS `.active`/`.fading`. ضغط الـ overlay = `skipIntroReveal()`.

## ٦. أفكار للتطوير القادم (اقتراحات لو إيمان طلبت "طوّرها أكتر")

أشياء ممكن تزوّد التعقيد/الجمال (نفّذها بس لو طلبت، وواحدة واحدة):
- **فجوات الحلقات (gaps):** دلوقتي الحلقات كاملة في الكانفس — أضف ثغرات بلون الخلفية
  زي العين الأصلية (استخدم `GAPS`) عشان تبان متاهة فعلية مش مجرد دوائر.
- **رسم الجفون progressive:** خليها ترسم عبر تتبّع جزئي للـ quadratic بدل ما تظهر
  كاملة، زي الـ stroke-dashoffset في العين الأصلية.
- **الحلزون الذهبي:** أضف لمسة ذهبية (Fibonacci bloom) كـ accent بسيط لحظة الذروة
  (بحذر — الذهبي لمسة، مش لون أساسي).
- **رمش/طرفة (blink):** طرفة واحدة خفيفة بعد ما العين تكمل، قبل الكلمة.
- **تتبّع المسار عبر الفجوات فعلاً:** خلي الـ solve path يعدّي من ثغرات الحلقات
  (يطابق مسار الحل الأصلي `#heroSolvePath`) بدل الحلزون العام.
- **عمق (parallax/glow layers):** طبقات توهّج إضافية تتنفّس بسرعات مختلفة.
- **صوت (لو يوماً اتطلب):** نقرة/رنين خفيف لحظة الوميض — بس ده بره النطاق الحالي.

**مبدأ:** خلي كل تطوير **متتابع وبطيء ومهيب**، حافظ على ice→cyan، وما تكسرش قاعدة
"مفيش filter على html/body". اختبر بـ Node بعد كل تغيير.
